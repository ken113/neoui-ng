Constructor

$( selector ).rate( options )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.rate" ).rate();
Options

defaultValue - 0

默认值, 取值0 - 5


selector4star - ">span"

Selector of the symbol


class4highlight - "highlight"

鼠标悬浮高亮样式


class4selected - "selected"

选中后样式


onSelect - null

选中后执行

$( ".ui.rate" ).rate( { onSelect: function( value ) { console.log( value ); } } );
Method

rate.val( [value] )

value - Number

不为空时设置为指定值(取值0 - 5), 为空时获取当前值



rate.disabled() - 禁用控件



rate.enabled() - 启用控件
