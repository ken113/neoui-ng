Example
{{ time || "00:00" }}




AngularJS
Markup
<s-timepicker ng-model="time" ng-disabled="isDisabled"> </s-timepicker>
Javascript
angular .module( "demo.timepicker", [ "$ui.timepicker" ] ) .controller( "timePickerController", [ "$scope", function( $scope ) { $scope.time = "11:11"; } ] );
Directive * s-timepicker

=ng-disabled - false

禁用控件


=ng-model - null

当前选择时间


&on-applied - null

选中时执行


@strict - "false"

所选时间必须存在于列表中


API
HTML
<div class="md-timepicker"> <input class="md-timepicker-hour" type="text" maxlength="2" value="00"> <span>:</span> <input class="md-timepicker-mintue" type="text" maxlength="2" value="00"> </div>


Constructor

$( selector ).timepicker( [options] )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".md-timepicker" ).timepicker();


Options

defaultTime - new Date()

默认时间


strict - false

为true时, 输入内容必须存在于列表中


onApplied - null

选中时间后执行




Method

.val( [ value ] )

value - String/Date

设置/取值


.disabled()

禁用控件


.enabled()

取消禁用
