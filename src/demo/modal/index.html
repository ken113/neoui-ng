Example


AngularJS
Service * $ui.modal
angular .module( "demo.modal", [ "$ui.modal", "$ui.message" ] ) .controller( "modalController", [ "$scope", "$modal", function( $scope, $modal ) { $scope.name = "test"; $scope.open = function( animation ) { return $modal.open( { controller: "modalController", animation: animation, templateUrl: "src/demo/modal/page.html", title: "Mango (Fruit)", class4modal: "demo", scope: $scope } ); }; $scope.showProgress = function() { $modal.open( { controller: "modalController", templateUrl: "src/demo/modal/page1.html", title: "弹出框标题 18PX 加粗 #333", class4modal: "demo", scope: $scope } ) .progress.start(); }; } ] );
Methods
$modal.open( options ) - 打开弹出框
options - Object

scope - null

指定scope

其他见Options


jQuery
Constructor
$.modal( options )
options - 参数设置
$.modal( { animation: "slide" } );
Options

title - "Modal.JS"

指定title

showTitle - true

显示标题头

modal - true

Modal / Modeless

render - null

弹出框内容, 字符串或function

draggable - true

true时, 通过标题栏拖拽, 做为字符串时作为选择器定位拖拽元素

class4modal - null

指定弹出框样式

autoShow - true

初始化后打开弹出框, 不通过实例open方法

closeByESC - true

Press Esc close the modal

closeByDocument - true

点击弹出框外面关闭弹出框

animation - "slide"

弹出框动画, slide, stick, bottom, flip, vertical, sign, scale, fadein

onClose - null

弹出框卸载触发

Methods
.open() - Open the modal

.close() - Close the modal

.progress - See Progress.js

.loading - See Loading.js
